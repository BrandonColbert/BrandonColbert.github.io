Tabs!

Tabs! is an extension for Google Chrome that enhances tab management capability. This functionality is introduced through dividers.

Dividers serve as a container for tabs. With the "Compress" feature found in the toolbar icon or context menu, a tab may be placed into a divider.

By saving websites to dividers, both clutter and memory consumption can be reduced.

Main Features

  • Divider list in toolbar menu
    • Divider reordering
    • Open dividers as tabs
    • Open a divider by full name or partial name
  • Divider tab
    • Batch compress all tabs to the left or right
    • Batch expand tabs
    • Search for a compressed tab
    • Create and organize subsections
    • Export as bookmark folder or to text file
  • Compresses tabs to nearby dividers with context menu
  • Change color theme and other options
