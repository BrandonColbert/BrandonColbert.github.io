Noise

Noise is a desktop application to create playlists from online videos.

Unlike traditional media players, Noise can compile videos from multiple web platforms into a single playlist. This is accomplished through a configurable, UI-driven pipeline system.

Plugins are also supported to enable additional functionality, such as visualizers, pop-out players, or ad-block.

Main Features

  • Player
    • Audio and video support
    • Playback controls
  • Playlist
    • Search for and start from song
    • Drag and drop song re-ordering
    • Grab link for song items
  • Descriptor Graph
    • Visually modify and create configurations
    • Configure how songs are found
    • Add support for additional websites
