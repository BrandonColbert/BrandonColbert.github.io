Supeer

Enables peer to peer communication in different contexts using WebRTC.

Supeer leverages WebRTC to create systems used to replicate the standard client-server model. This functionality enables connections between applications made in the browser, natively, or a combination of the two.

Main Features

  • Systems
    • Lobby
    • Signal Server
    • Bridge
    • Repeater
    • Proxy
  • Command Systems
    • Activate the systems
    • Provides help on accessing them
  • Launch shortcut to activate multiple commands at start
