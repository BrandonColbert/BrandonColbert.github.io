Formulas

Formulas is a library used to perform calculations based on text formatted in math notation.

Currently, Formulas fits two primary use cases.

Firstly, it can serve as a backend for a calculator to solve expressions consisting of numeric constants. These expressions may include basic operations as well as exponentiation, modulo, and parenthetical grouping.

Secondly, it can enable user-defined implementations of certain computations in an application. This is achieved by writing a function in the text whose inputs are used as variables in the function's definition.

Main Features

  • Solve expressions of consisting of constants
  • Solve functions involving any number of variables
  • Custom single-variable transformation functions such as sin, cos, and random
  • Type declaration on function inputs
  • Compilation of type specified functions to fast LINQ expressions
